# projectkinetica
A small physics simulation library for orbital mechanics with browser-native rendering. 


**4 languages used** Python, HTML, CSS, JavaScript

## What this simulation does 
- This simulator models any number of point mass bodies in 2d under mutual gravity
- Also integrates motion
- Exports each simulation to a single, self-contained HTML file that you can open in any browser.
  No wifi needed and no server.


## Files 
**orbital.py**: The purpose of this file is to act as the library of the simulation, this is where the orbital mechanics 
with browser-native rendering sits in. 
**Template.html**: The visualization template that incorporates HTML, CSS, and JS. This is the stripped down version of the simulation files 
**demo.py**: This is the three example simulations that you can run 
**earth_moon.html**: Earth-Moon system simulation 
**solar.html**: Inner solar system simulation that includes the Sun and 4 planets. 
**binary.html**: This is the binary star with a wider companion
